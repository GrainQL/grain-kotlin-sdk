# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Grain Kotlin SDK, please report it responsibly.

**Email:** security@grainql.com

Please include:
- A description of the vulnerability
- Steps to reproduce
- Any potential impact

We'll acknowledge your report within 48 hours and work with you to understand and address the issue before any public disclosure.

## Scope

This policy covers the Grain Kotlin SDK library code. For vulnerabilities in the Grain platform or backend services, please contact us at the same address.

## Supported Versions

Security fixes are applied to the latest release. We recommend always using the most recent version.
